# Midnight Sphere – Color System

## Brand Colors

### --primary
`#1060E9`
- Main brand color
- Logo color
- Primary CTA button
- Important highlights
- Active navigation item

### --primary-hover
`#0D51C4`
- Primary button hover
- Link hover
- Interactive states

### --accent
`#43A5F4`
- Rotating hero text
- Small highlights
- Gradient accents
- Aurora glow support

---

## Background Colors

### --bg
`#05070A`
- Main website background
- Hero section background
- Default page background

### --bg-secondary
`#0B1120`
- Alternate sections
- Feature sections
- Cards container areas

### --glass
`rgba(255,255,255,0.04)`
- Glassmorphism cards
- Navbar glass effect
- Floating code window

---

## Text Colors

### --text
`#FFFFFF`
- Main headings
- Important content

### --text-secondary
`#94A3B8`
- Paragraph text
- Sub-headings
- Descriptions

### --text-muted
`#64748B`
- Labels
- Metadata
- Small helper text

---

## Border Colors

### --border
`rgba(255,255,255,0.08)`
- Default borders
- Card outlines
- Input borders

### --border-accent
`rgba(16,96,233,0.25)`
- Hover borders
- Active cards
- Focus states

---

## Glow Colors

### --glow-blue
`rgba(16,96,233,0.35)`
- Main blue glow
- Hero spotlight effect

### --glow-light
`rgba(67,165,244,0.30)`
- Secondary glow
- Gradient support

---

## Huge Background Text

Use:
`rgba(255,255,255,0.03)`

Examples:
- MIDNIGHT
- SPHERE

Purpose:
- Watermark style background text
- Very subtle visibility

---

## CSS Variables

```css
:root {
  --primary: #1060E9;
  --primary-hover: #0D51C4;
  --accent: #43A5F4;

  --bg: #05070A;
  --bg-secondary: #0B1120;

  --glass: rgba(255,255,255,0.04);

  --text: #FFFFFF;
  --text-secondary: #94A3B8;
  --text-muted: #64748B;

  --border: rgba(255,255,255,0.08);
  --border-accent: rgba(16,96,233,0.25);

  --glow-blue: rgba(16,96,233,0.35);
  --glow-light: rgba(67,165,244,0.30);
}
```
